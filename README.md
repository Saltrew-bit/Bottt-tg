# AutoHub62 Bot

Бот для канала AutoHub62 с подачей объявлений и модерацией.

## Функции

- Приветствие при /start
- Кнопки: Подать объявление, Правила, Связь с админом
- Подача объявлений с фото
- Модерация админом
- Публикация объявлений в канал

## Установка

1. Клонировать репозиторий в ветку main.
2. Установить зависимости: `pip install -r requirements.txt`
3. Вставить токен в `bot.py`.
4. Запустить: `python bot.py`
