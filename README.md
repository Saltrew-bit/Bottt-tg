# AutoHub62Bot (webhook) для Bothost

1. Залейте репозиторий на Bothost.
2. В настройках Bothost укажите ветку `main`.
3. Установите переменную окружения `API_TOKEN` с токеном бота.
4. В настройках Bothost укажите endpoint `/bot`.
5. Запустите бот — /start сразу заработает, и панель будет гореть зелёным.
