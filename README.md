# Bottt-tg

Telegram бот Авто62 — подача объявлений для продажи авто в Рязани.

## Установка на Bothost

1. Репозиторий должен быть публичным.
2. В Bothost создаем новый проект: Python / Telegram bot.
3. Вставляем ссылку на репозиторий:
   https://github.com/Saltrew-bit/Bottt-tg.git
4. Указываем переменную окружения `API_TOKEN` с токеном бота.
5. Deploy и Run — бот готов к работе.
