# AutoHub62Bot for Bothost

1. Залейте репозиторий на Bothost.
2. Укажите ветку `main`.
3. В настройках Bothost укажите endpoint `/bot`.
4. Установите переменную окружения `API_TOKEN` с токеном бота.
5. Запустите бот — `/start` сразу заработает.
6. Бот поддерживает кнопки Подать объявление, Правила и Связь с админом.
